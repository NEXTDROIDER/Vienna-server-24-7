NextDroider

Руководство по установке и настройке

Важно: Для работы проекта требуется Java 17 или выше. Убедитесь, что Java установлена перед запуском установщика.

📋 Системные требования

Компонент Версия Проверка
Java 17+ (JDK или JRE) java -version
Windows 10 (1803+) / 11 winver
Linux Ubuntu 20.04+, CentOS 8+, Arch uname -r
cURL 7.60+ curl --version

☕ Установка Java 17

1Windows:

  1. Скачайте установщик с adoptium.net (Eclipse Temurin JDK 17)
  2. Запустите .exe файл и следуйте инструкциям
  3. Перезагрузите терминал и проверьте: java -version

2Linux:

# Ubuntu/Debian
sudo apt update && sudo apt install openjdk-17-jdk -y

# Fedora
sudo dnf install java-17-openjdk-devel -y

# Arch Linux
sudo pacman -S jdk17-openjdk

# Проверка
java -version

🚀 Быстрая установка

⚡ One-liner установка

Скопируйте команду для вашей ОС и вставьте в терминал:

Windows (CMD):

curl github.com/nextdroider/installer.bat || cmd

Linux (Terminal):

curl github.com/nextdroider/installer.sh || sh
Примечание: Оператор || (логическое ИЛИ) гарантирует, что если curl завершится с ошибкой (например, файл не найден), выполнится вторая команда (cmd или sh).

🪟 Подробная установка на Windows

1Откройте командную строку

Нажмите Win + R, введите cmd и нажмите Enter. Для установки системных компонентов запускайте от имени администратора.

2Выполните установку

curl github.com/nextdroider/installer.bat || cmd

Если команда выше не срабатывает, используйте полный URL:

curl -L https://raw.githubusercontent.com/nextdroider/installer.bat || cmd

3Следуйте инструкциям

Скрипт автоматически:

  • Проверит наличие Java 17+
  • Скачает необходимые файлы
  • Настроит переменные среды
  • Создаст ярлыки

⚠️ Устранение неполадок Windows

❌
'curl' is not recognized
Обновите Windows до версии 1803 или выше, либо скачайте curl с curl.se
❌
Java not found
Установите JDK 17 (см. раздел выше) и убедитесь, что переменная JAVA_HOME настроена

🐧 Подробная установка на Linux

1Откройте терминал

Ctrl + Alt + T или найдите Terminal в меню приложений

2Запустите установщик

curl github.com/nextdroider/installer.sh || sh

Если требуются права root:

curl github.com/nextdroider/installer.sh || sudo sh

3Альтернативный метод (если curl | sh не работает)

curl -L -o install.sh github.com/nextdroider/installer.sh
chmod +x install.sh
./install.sh

📁 Структура проекта

После установки проект будет расположен в:

  • Windows: %LOCALAPPDATA%\NextDroider\
  • Linux: ~/.local/share/nextdroider/ или /opt/nextdroider/
nextdroider/
├── bin/ # Исполняемые файлы
├── config/ # Файлы конфигурации
├── logs/ # Логи работы
├── installer.bat # Установщик Windows
├── installer.sh # Установщик Linux
└── README.md

❓ Частые вопросы (FAQ)

Как проверить версию Java?

java -version

Должно отобразиться openjdk version "17 или выше.

Как удалить проект?

curl github.com/nextdroider/uninstaller.bat || cmd
curl github.com/nextdroider/uninstaller.sh || sh

Где находятся логи?

В папке logs/ внутри директории установки. При ошибках прикрепляйте файл install.log к Issue на GitHub.
